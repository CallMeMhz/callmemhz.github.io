  5. 11. 盛水最多的容器

    Published: 周五 26 一月 2018
    By Mega Hertz

    In Leetcode.

    class Solution:
    def maxArea(self, height):
        """
        :type height: List[int]
        :rtype: int
        """
        l = 0; r = len(height) - 1;
        maxArea = 0;
        while l < r:
            maxArea = max(maxArea, min(height[l], height[r])*(r-l))
            if height[l] < height[r]:
                l += 1
            else:
                r -= 1
        return maxArea
